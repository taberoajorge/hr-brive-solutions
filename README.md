# HR Brive Solutions

Aplicación de gestión de recursos humanos.

## Tecnologías

- React
- Context API
- React Router

## Instalación

```bash
npm install
npm start
```

## Funcionalidades

- Lista de empleados
- Perfil de empleado
- Gestión de datos

## Propósito

Proyecto de práctica para manejo de estado con Context API.
